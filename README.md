# repositories-manager
Um gerenciador de repositórios do github.

### Pré-requisitos globais:
`npm i -g nodemon`

### Instalação
#### No diretório Server:
`cd server`
`npm install`
#### No diretório Client:
`cd ../client`
`npm install`

### Para rodar o projeto
#### No diretório Client:
`npm run start`
